
This project contains sample road accident data that requires basic cleaning operations such as:
- Handling missing values  
- Correcting inconsistent formats  
- Standardizing entries  

This documentation explains the purpose of the dataset and the cleaning steps that would normally be performed.

 Dataset Details
The dataset includes:
- Accident ID  
- Date  
- Location  
- Weather condition  
- Severity of accident  

Some values are missing or incorrectly formatted to simulate real-world raw data.

 Intended Cleaning Steps
1. Standardize all date formats  
2. Fill or remove missing values  
3. Ensure severity levels are consistent  
4. Correct incomplete entries
